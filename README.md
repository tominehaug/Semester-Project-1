# Science Museum Website

![image](https://user-images.githubusercontent.com/52622303/164316813-4b12d99f-aeb7-4069-85cf-e72b3a50ac99.png)

Website for a Science Museum! The product of my final semester project from my first semester of Front-End Development studies at Noroff Vocational School.

## Description

Design is self-made. Coding languages allowed in this project was HTML and CSS. 
The purpose of this project was to demonstrate practical skills and competence from courses in the first semester. The aim is to develop and publish a fully functional website with HTML and CSS.

## Technologies

- JavaScript
- HTML
- CSS
- ESLint
- Prettier
