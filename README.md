# Community Science Museum Website

![Community Science Museum project screenshot](https://raw.githubusercontent.com/tominehaug/portfolio/main/images/science-museum.jpeg)

Website for a Science Museum! The product of my final semester project from my first semester of Front-End Development studies at Noroff Vocational School.

## Description

Design is self-made. Coding languages allowed in this project was HTML and CSS. 
The purpose of this project was to demonstrate practical skills and competence from courses in the first semester. The aim is to develop and publish a fully functional website with HTML and CSS.

## Technologies

- JavaScript
- HTML
- CSS
- Prettier

## Getting Started

### Installing

1. Clone the repo:

```bash
git clone https://github.com/tominehaug/Semester-Project-1.git
```
Run this command in the Visual Studio Code Terminal.

2. Install the dependencies:

No dependencies for this project.

### Running

To run the app use LiveServer extension in Visual Studio Code.
